= Changelog

All notable changes to this project will be documented in this file.

The format is based on link:https://keepachangelog.com/en/1.0.0/[Keep a Changelog],
and this project adheres to link:https://semver.org/spec/v2.0.0.html[Semantic Versioning].

== [Unreleased]

=== Added

* Gleam-based backend with VeriSimDB integration on port 8100.
* AffineScript-based frontend using `affinescript-tea` pattern.
* `cadre-tea-router` adaptation for AffineScript compatibility.
* `gossamer.conf.json` for deployment orchestration.
* RSR (Rhodium Standard Repository) compliance documentation.
* Comprehensive security documentation (SECURITY.md).
* Community guidelines (CODE_OF_CONDUCT.adoc, CONTRIBUTING.adoc).
* Maintainer documentation (MAINTAINERS.adoc).
* .well-known directory for standards compliance.

=== Changed

* Migrated from Rails/Ruby to Gleam/AffineScript (typed-wasm) architecture.
* Replaced ReScript frontend with AffineScript.

=== Removed

* Legacy Rails application, database schema, and Ruby dependencies.

== [1.0.0] - 2025-01-22

=== Added

* Initial release of Candy Crash LMS platform (Legacy Rails implementation).
* User authentication with Devise.
* Course management system.
* Quiz engine.
* Progress tracking.
* Achievement system.
* Bootstrap 5 responsive UI.
